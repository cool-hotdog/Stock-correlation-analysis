# Pandas DataFrame 简介

## 什么是 DataFrame？

**DataFrame** 是 Pandas 库中最常用的数据结构。你可以把它极其简单地理解为一个 **Excel 表格**。

它的“样子”通常包含三个部分：
1.  **数据（Data）**：中间的具体数值或文本。
2.  **行索引（Index）**：最左侧的一列，用来标识每一行（通常默认是 0, 1, 2...）。
3.  **列名（Columns）**：最顶部的一行，用来标识每一列的特征。

### 示例图解

假设有一个名为 `df` 的变量：

|    | 姓名 | 年龄 | 城市 |
|---:|:---|:---|:---|
| **0** | Alice | 24 | 北京 |
| **1** | Bob | 30 | 上海 |
| **2** | Charlie| 28 | 广州 |

*   **0, 1, 2** 是 **行索引**。
*   **姓名, 年龄, 城市** 是 **列名**。

## 如何筛选数据？

在 Pandas 中，最常用的两种筛选方式是 `loc` 和 `iloc`。它们的区别在于**你是通过“名字”找，还是通过“位置”找**。

### 1. `loc`: 通过“标签/名字”筛选

**口诀：Look at Label (看标签)**

`loc` 使用行和列的**名称**来定位数据。

*   **语法**：`df.loc[行索引名称, 列名]`

**例子**：
想要获取 **Bob (索引为1)** 的 **年龄**：
```python
# 这里行的索引正好是数字 1，列名是字符串 "年龄"
print(df.loc[1, "年龄"])
# 输出: 30
```

想要获取 **所有行** 的 **"姓名"** 和 **"城市"**：
```python
print(df.loc[:, ["姓名", "城市"]])
```

### 2. `iloc`: 通过“整数位置”筛选

**口诀：Integer Location (整数位置)**

`iloc` 不管名字叫什么，只看它是**第几行、第几列**（从 0 开始数）。

*   **语法**：`df.iloc[行号, 列号]`

**例子**：
想要获取 **第 1 行 (Bob)**，**第 2 列 (城市)** 的数据：
*(注意：列的计数是 姓名=0, 年龄=1, 城市=2)*

```python
print(df.iloc[1, 2])
# 输出: 上海
```

想要获取 **前两行** 的数据：
```python
# 这里的 0:2 包含 0，不包含 2
print(df.iloc[0:2, :])
```

---

### 简单总结

*   如果你知道列名叫 "Age"，用 **`loc`**。
*   如果你只想取“最后一列”或者“第 3 列”，用 **`iloc`**。
